# Tastebin

A pastebin with taste i.e. minimalistic

* server: NodeJS, CoffeeScript, ExpressJS, ...
* client: HTML5, CoffeeScript, HighlightJs, ...

![SEEME](SEEME.png)

## Install and run

```sh
git clone git://github.com/andreineculau/tastebin.git
cd tastebin
npm install
# edit config.coffee as you see fit
npm start
```

## Tested browsers

* Chrome 44
